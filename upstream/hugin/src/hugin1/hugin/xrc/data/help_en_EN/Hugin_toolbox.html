Hugin toolbox

From PanoTools.org Wiki

Hugin_toolbox is a small helper program which allows to use the Hugin tools for several smaller task instead of calling the complete Hugin.

Enfuse GUI

The first part is a GUI for enfuse.

In the upper half you can load image files. Only the checked image files are used for the blending. You can also create brighter or darker versions of images for exposure fusion, which can be helpful for creating an image with a better brightness distribution.

In the lower half you can change the parameters of enfuse. See the enfuse manual for a more detailed description of these parameters. You can save and load a parameter set with Load/Save. The reset button resets the parameters back to default values.

The preview buttons does process the preview image in full resolution. To save the output use the output button (it is not necessary to process the preview before).

Perspective GUI

The second part is for do perspective correction on images (see e.g. Hugin tutorial on perspective correction)

After loading the image the focal length and crop factor is read from the EXIF data. The rotation of the input image can be selected by the choice box.

There are 2 mode for correction

  • Rectangle: Adjust the rectangle to an area which should be rectangular in the projected image.
  • Lines: Create a new line by dragging with left mouse button on a free space. Existing lines or line end points can be moved by dragging with left mouse button. A line can be deleted by clicking with the right mouse button. Lines can be created automatically with find lines button.

The crop of the output can be

  • outside: In this case the output contains (nearly) the complete input images, but it contains also masked areas without image information.
  • inside: The crop is so selected that no unused areas are in the output.

The output can be saved as TIF, JPEG or PNG image. For further tuning you can also save a pto project file and open this file then in Hugin.
